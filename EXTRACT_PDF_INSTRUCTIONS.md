# Instructions to Extract PDF Text

To extract text from the PDF files for varga chart calculations, run:

```bash
# Install pypdf if needed
pip install pypdf

# Or use the script
python3 extract_pdf_text.py
```

This will create:
- `chartcalculation.txt`
- `astrocalculation 2.txt`

Then you can review the text files to find the varga calculation methods and tables.
